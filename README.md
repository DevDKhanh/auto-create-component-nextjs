# Create Component

An extension to quickly generate a standard React component, including:

- `.tsx` file
- SCSS module file
- Interface
- Index exports

## How to use

- Right-click any folder in the Explorer
- Select **Create Component**
- Enter the component name

The extension will automatically generate all necessary files.
